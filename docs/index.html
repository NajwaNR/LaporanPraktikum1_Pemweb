Jawaban Teori Pemrograman Web B

1. Pertanyaan seputar tag

  1. HTML (Hypertext Markup Language) adalah bahasa yang dipakai dalam menyusun struktur sebuah halaman web dengan cara marking up setiap bagian kontennya, nah penanda ini bernama Tag. Tag ditulis menggunakan dua tanda kurung sudut (< >), yang memberitahu browser bagaimana menampilkan suatu konten. Contoh tag yang sering digunakan: <h1>, <p>, <a>, <img>. Tag memberi tahu website misal bagian yang diapit <p> dan </p> adalah bagian paragraf, <h1> dan </h1> adalah headingnya.
  2. Tag memiliki 2 jenis, pertama Tag Berpasangan (Paired Tags), yang memiliki tag pembuka dan tag penutup. Tag pembuka ditulis biasa seperti <p>, sedangkan tag penutup ditambahkan tanda garis miring, seperti </p>. Contoh lainnya ada <h1>Judul Besar</h1>. Jenis tag kedua adalah Tag Tunggal (Empty Tags) yang tidak memiliki tag penutup dan langsung berdiri sendiri seperti <img>, <br>, dan <input>. Tag-tag ini tidak membutuhkan tag penutup karena tidak memiliki konten di dalamnya.

2. Pertanyaan seputar element

  1. Setiap halaman web yang dibuka di browser merupakan dokumen HTML yang tersusun dari element-element. Element adalah unit lengkap dalam HTML yang terdiri dari tag pembuka, konten (isi), dan tag penutup. Contohnya <p>Ini adalah sebuah paragraf.</p> sudah menjadi satu element paragraf yang lengkap.
  2. Tagname adalah nama dari tag yang menentukan jenis elemen. Misalnya pada <a href="...">, tagname-nya adalah a (anchor/tautan). Lalu Attribute adalah informasi tambahan yang diberikan di dalam tag pembuka untuk mengatur perilaku atau tampilan elemen. Contohnya pada <img src="gambar.jpg" alt="Deskripsi gambar">, atributnya adalah src (lokasi file gambar) dan alt (deskripsi gambar). Kemudian Content adalah isi yang berada di antara tag pembuka dan tag penutup. Bisa berupa teks, gambar, atau elemen HTML lain yang bersarang di dalamnya.
  3. Element HTML dapat ditumpuk, misal element <body> bisa memuat elemen <h1> dan <p> di dalamnya, dan <header> bisa memuat <nav> yang memuat <ul> yang memuat <li> didalamnya (Tumpuk tiga). Saat elemen ditumpuk, elemen luar menjadi parent dan elemen dalam menjadi child. Penumpukan harus dilakukan dengan urutan yang benar, yaitu tag yang dibuka terakhir harus ditutup terlebih dahulu agar browser bisa merender tampilan dengan benar.

3. Pertanyaan seputar anchor

  1. Anchor adalah elemen HTML yang digunakan untuk membuat hyperlink ke halaman lain, file, atau lokasi tertentu. Contohnya <a href="https://www.google.com">Kunjungi Google</a> akan mengarahkan ke halaman Google.com.
  2. Anchor didefinisikan menggunakan tag <a> dengan atribut href untuk menentukan URL tujuan, serta atribut target="_blank" jika ingin membuka tautan di tab baru. Dalam pengaplikasiannya, elemen ini digunakan untuk membuat tautan eksternal ke situs lain, tautan internal ke bagian tertentu dalam halaman yang sama menggunakan ID elemen.

4. Pertanyaan seputar multimedia

A. Text

  • Tag HTML seperti <b>, <i>, <u>, dan <mark> digunakan untuk memformat teks (tebal, miring, garis bawah, sorot). Tag <sup> dan <sub> untuk teks kecil di atas dan bawah, sedangkan <span> untuk mengatur bagian teks secara inline dengan CSS.
  • Tag <h1>–<h6> untuk judul, <p> paragraf, <br> baris baru, <hr> garis, <ul>/<ol> dan <li> untuk daftar, serta <div> untuk mengelompokkan konten.

B. Image

  • Penyebab Gagal adalah bisa karena Path atau URL pada atribut src salah, atau format atau ekstensi file gambar tidak sesuai, bisa juga karena file gambar rusak atau format gambar tidak didukung oleh browser yang digunakan.
  • Teks pengganti gambar berasal dari atribut alt pada tag <img>, yang akan ditampilkan saat gambar gagal dimuat dan juga digunakan oleh screen reader.

C. Audio

  • Memakai tag <audio> dengan atribut controls agar browser menampilkan tombol play, pause, dan volume, kemudian di dalam tag <audio>, tambahkan tag <source> dengan atribut src yang diisi URL langsung menuju file audio di internet (non-lokal).

D. Video

  • Browser modern memblokir autoplay dengan suara tanpa interaksi pengguna. Autoplay akan diizinkan jika video dalam kondisi muted, atau setelah pengguna berinteraksi, misal <video autoplay muted controls>.
